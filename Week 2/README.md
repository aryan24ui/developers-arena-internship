# 🌐 Styled Portfolio Website

## 📖 Project Overview

This project is a responsive **Personal Portfolio Website** developed using **HTML5 and CSS3** as part of **Week 2 – CSS Styling Your Website** in The Developers Arena Internship Program.

The portfolio showcases my personal profile, technical skills, and contact information through a clean, modern, and user-friendly interface. The primary goal of this project was to transform a basic HTML portfolio into a visually appealing website by implementing various CSS styling techniques.

The website demonstrates the practical application of CSS concepts such as selectors, colors, typography, box model, Flexbox, hover effects, transitions, media queries, and responsive design. Special attention was given to improving layout structure, visual consistency, user experience, and mobile compatibility.

---

## 🎯 Project Objectives

- Create a visually appealing portfolio website.
- Apply CSS styling to an HTML webpage.
- Implement responsive design for different screen sizes.
- Use Flexbox for layout management.
- Add hover effects and smooth transitions.
- Improve user experience through modern UI design.
