# 🚀 Interactive Portfolio Website

An interactive and responsive personal portfolio website developed using **HTML5, CSS3, and JavaScript** as part of **Week 3 – JavaScript: Making Websites Interactive** in The Developers Arena Internship Program.

---

# 📖 Project Overview

This project transforms a static portfolio website into a fully interactive web application by implementing JavaScript concepts such as:

- DOM Manipulation
- Event Handling
- Form Validation
- Dynamic Content Updates
- Dark/Light Mode
- Interactive UI Components

The website showcases my profile, skills, and contact information while providing an improved and engaging user experience.

---

# 🎯 Project Objectives

✅ Create an interactive portfolio website.

✅ Apply JavaScript to add dynamic functionality.

✅ Implement form validation with proper error handling.

✅ Use DOM manipulation techniques.

✅ Add multiple interactive features to improve user experience.

✅ Build a responsive and user-friendly interface.
